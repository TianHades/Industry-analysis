# 产能周期判断法 — 成熟期"周期化"的放大镜

## R — 原文 (Reading)

> 成熟期行业中有一类"稳定市场周期化"的行业（如钢铁、有色的采掘与冶炼、工程机械），它们会呈现出显著的产能周期——供不应求→产能激增→产能出清→格局改善。判断产能周期位置最可靠的指标之一是 CAPEX ÷ 折旧摊销 < 1.5 时，行业可能处于底部区域。
>
> — 肖璟, 第5章

---

## I — 方法论骨架 (Interpretation)

### 产能周期的四个阶段

```
阶段1：供不应求 → 产能利用率高，厂商开始扩产
阶段2：产能激增 → 大量竞争者涌入，供过于求，价格下跌
阶段3：产能出清 → 弱者退出，行业悲观，供给收缩
阶段4：格局改善 → 供需恢复平衡，龙头份额提升
```

### 产能扩张三阶段定性判断法

| 阶段 | 信号 | 含义 |
|------|------|------|
| ① 龙头被动扩产能 | 现有产能不够用，添置设备/加班 | 需求旺盛，行业健康 |
| ② 龙头主动扩产能 | 看好长期前景，开新厂/招人 | 行业信心足，但可能过热 |
| ③ 外来者无序扩产能 | 行业外资本涌入，跟风投产 | ⚠️ **行业顶部信号**！离产能过剩不远 |

> **辅助判断**：行业底部通常出现在龙头减产、本赛道玩家退出、卖方分析师也不再推荐之时；行业顶部往往出现在龙头扩产、行业外玩家疯狂涌入、卖方分析师跳空式上调评级之时。

### CAPEX 定量分析法

资本性支出（CAPEX）是产能扩张的前瞻性指标：

```
CAPEX 增加 → 在建工程增加 → 固定资产增加 → 产能释放
```

- CAPEX = 购建固定资产等支付的现金 − 处置固定资产等收回的现金
- 更准确的计算：CAPEX = 净经营长期资产增加 + 折旧与摊销

**关键判断信号**：当 **CAPEX ÷ 折旧摊销 < 1.5**（长江证券经验阈值），说明新增产能投入有限，可能处于供给收缩、格局即将改善的底部区域。比率跌破 1 意味着几乎没有新增投资。

### 两种特殊情况

| 情况 | 描述 | 示例 |
|------|------|------|
| **龙头逆势扩张** | 行业需求下行，但龙头凭借融资优势逆势扩产，强化规模优势 | 行业寒冬后的格局改善红利 |
| **龙头主动价格战** | 行业需求无问题，但龙头主动降价压低毛利率，淘汰中小玩家 | 2022年底新能源汽车价格战、2000年前后空调行业 |

> 这两种情况下，短期利润承压但长期集中度将大幅提升，值得关注被错杀的龙头。

### 库存周期（短期供需的"蓄水池"）

```
主动补库存 → 被动补库存 → 主动去库存 → 被动去库存
（乐观备货）   （卖不动积压）  （打折甩卖）   （需求恢复库存减少）
```

**关键判断信号**：
- 库存达到**历史高位** → ⚠️ 供过于求的危险信号
- 库存达到**历史低位** → 市场供不应求或已进入磨底阶段

> **汽车行业示例**：汽车经销商库存系数 = 期末库存量 ÷ 当期销售量。该系数 ≥ 2.5 时，经销商经营压力大，产能短期不会扩张。中国汽车流通协会每月发布该数据。

## A1 — 书中的应用 (Past Application)

### 案例 1: 小明面馆的产能周期（微观视角）
- **问题**: 一条街上只有一家面馆时生意好，后来开了8家，大家都不赚钱
- **方法论的使用**: 产能周期四阶段完整呈现——1家（供不应求）→ 8家（产能激增，供过于求）→ 有些面馆倒闭（产能出清）→ 剩下3家各有特色，生意稳定（格局改善）
- **结论**: 产能周期的本质是供给端的自我调节——过剩→出清→再平衡
- **结果**: 最终留下的面馆盈利能力恢复

### 案例 2: 空调行业价格战（龙头主动价格战）
- **问题**: 2001-2004年空调行业价格战，全行业利润大幅缩水
- **方法论的使用**: 格力/美的作为龙头主动降价→压低全行业毛利率→中小品牌无法承受亏损→退出市场→CR3从37%升至55%
- **结论**: 这是典型的"龙头主动价格战"——短期利润承压但长期集中度大幅提升
- **结果**: 价格战后格力美的成为双寡头，长期超额回报

### 案例 3: 煤炭行业的"回光返照"行情
- **问题**: 煤炭行业被认为是衰退期/夕阳行业，但仍有阶段性的投资机会
- **方法论的使用**: 产能出清阶段 → 供给收缩速度快于需求下降速度 → 供需短暂失衡 → 价格反弹。"回光返照"行情的本质是供给收缩而非需求复苏
- **结论**: 衰退期行业可能存在供给侧驱动的阶段性机会，但需严格区分"供给收缩"和"需求复苏"
- **结果**: 在正确的产能出清阶段入场可获得阶段性超额收益

### 语言信号

- "库存高不高？"

- "龙头在扩产还是在减产？"
- "CAPEX/折旧比是多少？"
- "是不是到底部了？"

### 与相邻 skill 的区分

- 与 `industry-lifecycle-anchoring` 的区别: 产能周期是成熟期"稳定市场周期化"内的**子周期**，一个行业可能在成熟期内经历多轮产能周期。生命周期定位是宏观导航，产能周期是微观探针。
- 与 `prosperity-tracking-system` 的区别: 景气度追踪是**持续验证**工具，产能周期判断是**一次性定位**工具。

## E — 可执行步骤 (Execution)

1. **定性判断产能扩张阶段**
   - 搜索行业新闻判断是"被动扩/主动扩/外来者无序扩"中的哪个阶段
   - 搜索执行参考: `reference/search-specs.md`
   - 完成标准: 给出明确的扩张阶段 + 信号依据

2. **定量分析CAPEX**
   - 获取主要上市公司的CAPEX和折旧摊销数据（优先AkShare）
   - 计算 CAPEX ÷ 折旧摊销 比率
   - 完成标准: 给出龙头企业的CAPEX/折旧比 + 与1.5经验阈值的比较
   - 判停条件：非上市公司主导的行业，用行业宏观产能数据替代

3. **判断库存周期位置**
   - 搜索行业库存数据（库存量/库存系数/库存天数）
   - 与历史水位对比
   - 完成标准: 明确"主动补/被动补/主动去/被动去"阶段

4. **识别特殊情况**
   - 是否存在龙头逆势扩张或龙头主动价格战？
   - 完成标准: 对每种特殊情况给出"存在/不存在"的判断

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 导入期或成长期行业——产能周期分析专属于成熟期"稳定市场周期化"行业
- 轻资产/非制造业行业（如互联网、金融、咨询）——产能和库存的概念在这些行业缺乏实质意义
- 用户只想确认当前的产能利用率数字——这是数据查询，不是周期判断

### 作者在书中警告的失败模式

- **外来者无序扩产是顶部信号**：当行业外玩家（如网吧老板跨界开面馆）开始涌入时，行业顶部通常不远了——这是被反复验证的产能周期见顶信号
- **市盈率误用于强周期股**：对强周期股用市盈率会买在利润高点——应该用CAPEX/折旧比等周期指标
- **库存高位≠马上跌**：库存达到历史高位是危险信号，但不代表明天就会跌——去库存可能持续几个季度

### 作者的盲点 / 时代局限 (来自 BOOK_OVERVIEW.md)

- **制造业中心视角**：产能/库存周期框架高度锚定制造业和资源行业，对服务业和数字经济的适用性有限
- **A股经验阈值**：CAPEX/折旧 < 1.5 的长江证券经验阈值在A股有效，其他市场需验证
