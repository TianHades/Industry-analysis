# 行业定义与边界 — 分析的起点

## R — 原文 (Reading)

> 行业研究的第一步是搞清楚"我在分析什么"——行业的定义是什么、边界在哪里、和相似行业有什么区别。
> 产业链是行业内部的纵向结构——上游、中游、下游各环节的利润分配往往比同一环节内企业间的差异更大。
>
> — 肖璟, 第1章 + 第5章

---

## I — 方法论骨架 (Interpretation)

### 行业定义的三要素

1. **标准定义**：一句话说清这个行业是什么
2. **行业分类**：在标准分类体系中的位置（申万/中信/GICS/国民经济行业分类）
3. **边界判断**：什么算这个行业、什么不算？和相似行业的区别？

### 多子赛道行业处理

有些行业包含多个处于不同生命周期阶段的子赛道（如教培行业：K12→衰退期、素质教育→成长期、职业教育→成熟期），**应拆分为子赛道分别判断**，不要强行给出单一结论。

### 核心特征速写

| 特征维度 | 说明 |
|---------|------|
| 高频/低频 | 用户消费频率 |
| 刚需/弹性 | 需求对价格的敏感度 |
| 成瘾性 | 用户是否有重复使用/消费的内在动力 |
| 周期性 | 是否受宏观经济周期显著影响 |

### 产业链结构（纵向分析）

```
上游：原材料/核心零部件 → 中游：生产/制造/服务 → 下游：渠道/终端用户
```

产业链分析要点：
- 各环节的主要参与者和集中度
- 各环节的毛利率水平和利润分配
- 利润在哪个环节最集中？哪个环节被挤压？
- **关键洞察**：产业链各环节的利润水平差异往往比同一环节内企业间的差异更大——选择正确的产业链环节比选择正确的企业更重要

## A1 — 书中的应用 (Past Application)

### 案例 1: 新能源汽车产业链拆解
- **问题**: 新能源汽车行业范围太广，怎么确定分析范围？
- **方法论的使用**: 拆解为上游（锂矿/钴矿/正负极材料）→ 中游（动力电池/电机/电控）→ 下游（整车制造/充电桩/后市场）。2020年分析时，中游电池环节确定性最高（格局初步清晰、技术壁垒高），上游矿资源波动大，下游整车不确定性高
- **结论**: 产业链拆解帮助投资者找到了当时最具投资价值的环节——中游电池
- **结果**: 2020-2022年中游电池企业获得最大涨幅

### 案例 2: 教培行业的多子赛道拆分
- **问题**: 2021年"双减"政策后，教培行业还能分析吗？
- **方法论的使用**: 不能简单说"教培行业完蛋了"——K12学科培训进入衰退期，但素质教育（编程/艺术/体育）处于成长期，职业技能培训处于成熟期，教育信息化仍有政策支持
- **结论**: "教培行业"是一个笼统标签，需要拆分为子赛道各自分析——简单粗暴的"行业死亡"判断会错过成长期子赛道的机会
- **结果**: 素质教育、职业教育赛道在政策冲击后逐步恢复增长

## E — 可执行步骤 (Execution)

1. **确定行业定义与分类**
   - 搜索行业标准定义（优先：国家统计局/行业协会/GICS/申万分类标准）
   - 搜索执行参考: `reference/search-specs.md`
   - 完成标准: 标准定义 + 行业分类代码 + 与至少1个相似行业的区分

2. **判断是否需要拆分**
   - 检查行业是否包含多个处于不同生命周期阶段的子赛道
   - 如果需要拆分，列出子赛道清单并分别标注生命周期阶段
   - 完成标准: 单一行业或子赛道清单

3. **绘制产业链结构图**
   - 识别上中下游环节和主要参与者
   - 收集各环节毛利率/利润率数据
   - 完成标准: 产业链结构图 + 利润分配表

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 行业定义和边界已经明确——如果用户和分析师对"什么是这个行业"有共识，不需要在此模块花时间
- 只需要产业链内某一个环节的深度分析——此时应直接进入对应子技能（如竞争格局、护城河）
