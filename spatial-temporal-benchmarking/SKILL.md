# 时空对标推理法 — 用过去和他处理解当下

## R — 原文 (Reading)

> 时间对标：回顾历史，看过去有没有类似的需求被成功满足过。底层逻辑——人性需求不变。
> 空间对标：对标成熟市场（如美国、日本），看有没有类似模式成功过。底层逻辑——"人类的底层需求是相通的，成熟市场的经验可作参考"。
>
> 搭建UE模型要看终局、判断行业集中度要看成熟市场对标、估算市场规模要看稳态TAM——所有分析都要有"终局感"。
>
> — 以上内容综合自肖璟，第2章及全书多处终局思维方式

---

## I — 方法论骨架 (Interpretation)

时空对标推理法是用**过去**和**他处**的经验来理解**当下**和**此处**的推理框架。它弥补了演绎法在信息不完备时的推理盲区——当你无法从第一性原理推导出结论时，类比是最实用的替代推理路径。

### 时间对标（纵向推理）

**核心逻辑**：人类的基本需求（衣食住行、社交、安全、自我实现）几千年没有变过。今天看起来"全新"的商业模式，往往可以在历史上找到满足同类需求的"前世"。

**操作方法**：
1. 抽象出当前行业/模式满足的**底层需求**（不是产品形态，而是用户需求）
2. 在历史上寻找满足同类需求的产品/服务/制度
3. 分析历史案例的演化路径和最终结局
4. 推演当前行业可能的演化方向

**示例**：
- 共享单车 → 底层需求是"短途出行" → 历史上自行车租赁、公共自行车系统
- 直播电商 → 底层需求是"互动式购物体验" → 历史上电视购物
- 知识付费 → 底层需求是"自我提升" → 历史上书籍出版、培训班

### 空间对标（横向推理）

**核心逻辑**：不同国家/地区的经济发展阶段不同，但产业发展路径高度相似。"时光机理论"——发达市场的今天是发展中市场的明天。

**操作方法**：
1. 选择一个处于更成熟阶段的参照市场（通常是美国、日本、欧洲）
2. 找到该市场中对应行业的渗透率、集中度、利润率等关键指标
3. 分析参照市场的产业发展路径（经历了哪些阶段、关键转折点）
4. 以此为标杆，推演当前市场可能的路径

**⚠️ 空间对标的有效性前提**（使用前必须验证）：
- 文化可比性：两地的消费习惯、社会结构是否相似？
- 制度可比性：两地的政策环境、监管框架是否相似？
- 经济可比性：两地的经济发展水平差距是否在合理范围？
- 如果以上三项中任何一项不满足，空间对标的结论需要显著打折

### 终局对标（前瞻性推理）

**核心逻辑**：所有分析都要有"终局感"——不是看行业当下是什么样，而是看行业稳态下应该是什么样。

**终局对标的三类问题**：
1. **终局UE模型**：当行业完全成熟后，单店/单车/单客的单位经济模型是什么样？
2. **终局集中度**：当行业进入稳态后，CR3/CR5 大概是多少？
3. **终局TAM**：当渗透率达到天花板后，市场总规模大概是多少？

**操作方法**：
- 找到最成熟的参照市场（通常是空间对标的结果）
- 提取该市场当前的稳态指标
- 判断当前市场向终局收敛的路径和速度

## A1 — 书中的应用 (Past Application)

### 案例 1: C2C（Copy to China）模式的空间对标
- **问题**: 2000-2010年代，大量中国互联网创业项目直接"复制"美国成功模式（百度→Google、淘宝→eBay、微博→Twitter），如何判断哪个模式能在中国成功？
- **方法论的使用**: 对每个美国模式做空间对标的三步验证——(1) 文化可比：中美用户的使用习惯有何差异？(2) 制度可比：中国的监管环境有何不同？(3) 经济可比：中国的支付/物流/信用基础设施是否支撑该模式？
- **结论**: 简单照搬美国模式的失败率很高（eBay在中国被淘宝打败），但经过本地化调整的模式成功率大幅提升
- **结果**: 那些不只是"Copy"而是"Copy + Adapt"的企业（如淘宝做担保交易、微信做红包）最终胜出

### 案例 2: 共享篮球——时间对标失败的案例
- **问题**: 2017年共享经济风口下，共享篮球项目出现——扫码租篮球，用后归还
- **方法论的使用**: 时间对标——"打篮球忘了带球"这个需求历史上是怎么被满足的？答案是"借朋友的球"或"球场边小店租球"。这两个方案的成本都极低（社交成本/几元租金），共享篮球的押金+扫码流程反而更复杂
- **结论**: 时间对标揭示了共享篮球的伪需求——用户根本没有被"借不到球"这个痛点困扰过。历史上低成本方案已经充分满足了这个需求
- **结果**: 共享篮球项目迅速失败

### 案例 3: 纸巾行业的终局对标（日本→中国）
- **问题**: 中国纸巾行业在成长期后格局会如何演变？
- **方法论的使用**: 空间对标日本纸巾行业——日本纸巾行业CR3高达60%+，且集中在少数几个品牌手中。中国纸巾行业的发展路径（品牌化→集中度提升→高端化）高度类似
- **结论**: 中国纸巾行业在成长期后必然走向集中度提升，龙头企业的护城河会越来越宽
- **结果**: 中顺洁柔、维达等龙头企业确实在行业成熟后获得了持续的超额收益

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. 用户看到一个新商业模式，想判断"这个模式靠不靠谱"——用时间对标检查历史上有没有类似需求被成功满足过
2. 用户想了解一个行业的发展空间——"中国XX行业的终局大概是什么样？"——用空间对标参照成熟市场
3. 用户在搭建财务模型时需要终局假设——终局毛利率、终局费用率、终局市场份额
4. 用户在比较两个市场的差异——"为什么XX模式在美国成功了但中国不行？"

### 语言信号

- "对标美国/日本/欧洲市场"
- "这个行业的终局会是什么样？"
- "历史上有没有类似的？"
- "时光机理论"
- "为什么国外能成功？"
- "天花板在哪里？"
- "稳态下会是什么格局？"

### 与相邻 skill 的区分

- 与 `hypothesis-driven-research` 的区别: 时空对标是**类比推理**（用过去/他处理解当下），假设驱动是**演绎验证**（先假设再证明）。两者是互补的推理路径——优先级取决于信息可得性：有成熟参照市场时优先用时空对标，无参照时优先用假设驱动。
- 与 `business-model-feasibility` 的区别: 可行性验证是时空对标的具体应用场景之一——时间对标和空间对标是可行性验证的两大核心工具。
- 与 `market-size-estimation` 的区别: 市场规模测算常用终局对标来做锚定——"成熟市场的TAM是X，所以我们的TAM大约是Y"。

## E — 可执行步骤 (Execution)

1. **确定对标目的**
   - 明确是哪种对标：时间对标（验证需求真实性）、空间对标（预判发展路径）、还是终局对标（测算天花板）
   - 完成标准: 明确定义"我们要对标什么、为了回答什么问题"

2. **选择对标参照物**
   - 时间对标：抽象底层需求 → 搜索历史上满足类似需求的产品/服务
   - 空间对标：选择1-2个最接近的成熟市场（优先：美国、日本、欧洲）
   - 终局对标：找到该市场当前稳态下的关键指标（渗透率、集中度、利润率、UE模型）
   - 完成标准: 每个对标维度找到至少1个参照物，标注参照物的可比性

3. **验证可比性前提**
   - 文化可比性：消费习惯、社会结构
   - 制度可比性：政策环境、监管框架
   - 经济可比性：经济发展水平差距
   - 完成标准: 产出可比性检查清单，标注每个维度的"可比/差异大/不可比"
   - **判停条件**: 如果三项中有两项"差异大"或任一项"不可比"，空间对标的结论需要显著打折，标注"🔍 对标有效性有限"

4. **构建对标分析**
   - 列出参照物的发展路径（关键阶段、转折点、时间跨度）
   - 比较当前行业与参照物的同阶段差异
   - 预测当前行业的关键转折点时间窗口
   - 完成标准: 产出"当前行业 vs 参照行业"的差异分析表 + 发展路径推演

5. **终局建模**
   - 基于对标结果，搭建终局UE模型/终局竞争格局/终局TAM
   - 区分"确定性的终局特征"（如集中度上升）和"不确定性的终局特征"（如具体时间）
   - 完成标准: 至少给出终局状态的关键参数（渗透率天花板、稳态CR3、稳态利润率）

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **极度新兴、无任何参照的行业**（如太空采矿、脑机接口）：时空对标法会完全失效——没有历史参照、没有成熟市场参照。此时只能使用 hypothesis-driven-research 的第一性原理推理。
- **文化/制度差异大到对标失去意义的跨市场比较**：如用美国医疗行业对标中国医疗行业（制度差异巨大），对标结论几乎没有参考价值。
- **用户只需要一个简单数据而非推理框架**：如"日本纸巾行业CR3是多少"是数据查询，不需要启动对标推理。
- **纯技术创新领域的"终局"判断**：技术路线的不确定性可能让终局对标变成"刻舟求剑"。

### 作者在书中警告的失败模式

- **时光机理论的适用性退化**（来自 BOOK_OVERVIEW.md 批判）：空间对标的底层假设是"人类底层需求相通+成熟市场经验可复制"。在逆全球化和本土化兴起的时代，跨市场经验的可复制性在下降。必须验证可比性前提后再使用。
- **表面对标、忽略深层差异**：只看到日本纸巾CR3=60%就预测中国也会是60%，但忽略了两个市场在渠道结构、消费习惯、政策环境上的差异——这些差异可能让终局集中度完全不同。
- **时间错配**：用20年前的美国对标今天的中国——两个市场的经济发展阶段不同，不能用同一时间截面对标。

### 作者的盲点 / 时代局限 (来自 BOOK_OVERVIEW.md)

- **A股/中国中心视角**：书中对标案例以A股为主，对全球投资者的对标需求覆盖不足
- **发达国家→发展中国家的单向对标**：忽略了反向对标——有些领域中国已经领先（如移动支付、短视频），应该是他处对标此处而非相反

### 容易混淆的邻近方法论

- **不做简单的X=Y**: "日本纸巾CR3=60% → 中国也会是60%"这不是对标分析，这是抄答案。真正的对标分析要分析"为什么日本是60%、中国的哪些条件会导致最终数据偏离这个标杆"。
- **终局对标 ≠ 预测未来**: 终局对标给出的是"如果前提不变，最可能的稳态"——一旦外部前提变化（政策、技术突破、社会变迁），终局需要重新对标。
