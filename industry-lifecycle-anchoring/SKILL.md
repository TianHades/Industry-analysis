# 产业生命周期定位术 — 整个框架的"锚"

## R — 原文 (Reading)

> 渗透率描述的是行业已经触达了多大比例的潜在用户……一旦行业的渗透率达到15%~20%，行业就会进入快速提升的阶段，此时对应的是行业的成长期。但是渗透率提升至35%~40%以后，行业发展的脚步往往就开始放慢了，此时对应的是行业的成熟期。在现实世界里，行业的发展也不像传统的产业生命周期呈现的那样，有着统一的S形曲线，而是在每一个阶段都有着不同的可能性。
>
> — 肖璟, 第1章

---

## I — 方法论骨架 (Interpretation)

产业生命周期是整个行业研究框架的**锚**——它决定了"现在什么分析维度最重要"。核心工具是**渗透率**（存量用户 ÷ 潜在客户群），两个关键临界点：**15%~20%**（导入期→成长期）和**35%~40%**（成长期→成熟期）。

### 四个阶段的特征与分析重点

```
导入期 → 核心问题："这事靠不靠谱？"
  ├─ ★★★ 商业模式可行性（最重要！验证需求真伪和盈利可能）
  ├─ ★★★ TAM潜在市场规模（判断天花板够不够高）
  ├─ ★★☆ 行业定义、产业链结构（基础认知）
  ├─ ★☆☆ 竞争格局（还没什么竞争者）
  └─ ☆☆☆ 护城河、估值（为时过早）

成长期 → 核心问题："天花板有多高？"
  ├─ ★★★ 市场规模 TAM/SAM/SOM（最核心！增速和空间）
  ├─ ★★★ 产业生命周期（判断拐点，渗透率是关键）
  ├─ ★★☆ 竞争格局（玩家增多，格局初现）
  ├─ ★★☆ 商业模式（已被验证，但仍需关注UE模型）
  ├─ ★☆☆ 护城河（开始形成，但非重点）
  └─ ★☆☆ 估值（PEG/PS法，波动大）

成熟期 → 核心问题："谁能胜出？怎么分蛋糕？"
  ├─ ★★★ 竞争格局 CRn（最核心！集中度决定利润率）
  ├─ ★★★ 护城河（深度分析，壁垒是否可持续）
  ├─ ★★☆ 估值（PE/PB法，关注分红和回购）
  ├─ ★★☆ 景气度跟踪（周期波动影响大）
  ├─ ★☆☆ 商业模式（早已验证，无需重点分析）
  └─ ★☆☆ 市场规模（增速放缓，天花板已近）

衰退期 → 核心问题："有没有第二春？"
  ├─ ★★★ 第二增长曲线/替代品分析
  ├─ ★★★ 估值（警惕低估值陷阱）
  ├─ ★★☆ 景气度跟踪（确认下行趋势）
  └─ ★☆☆ 护城河（可能在瓦解）
```

**执行规则**：
1. **先判断生命周期阶段**，再根据阶段决定各模块分析深度
2. ★★★ 模块必须详细分析，★★☆ 适度展开，★☆☆ 简要带过，☆☆☆ 可以跳过
3. **不要平均用力**——面面俱到的报告 = 没有重点的报告
4. ⚠️ **行业特性例外**：上述优先级矩阵是通用指引，不可刻舟求剑。当行业特性与矩阵显著冲突时（例如平台型成长期企业的网络效应护城河极其关键、不应评为★☆☆），应以行业实际为准，并在报告中说明调整原因

### 渗透率的理论基础：创新扩散理论

渗透率之所以能划分产业生命周期，底层逻辑来自 E.M.罗杰斯的创新扩散理论——不同人群接受创新的速度不同：

| 用户类型 | 特征 | 典型行为 |
|---------|------|----------|
| **创新者** (2.5%) | 好奇心强、容错率高、愿意尝鲜 | 首批特斯拉车主 |
| **早期采用者** (13.5%) | 圈内 KOL/KOC，有影响力 | 紧跟首批车主并在社交平台分享体验 |
| **早期大众** (34%) | 深思熟虑但心态开放 | 看完评测、试驾后从 BBA 转向国产电动车 |
| **后期大众** (34%) | 顾虑多、跟风购买 | 等身边朋友验证后才敢入手 |
| **落后者** (16%) | 因循守旧、信息闭塞 | 燃油车彻底不行了才考虑新能源 |

产业生命周期的本质就是产品不断"破圈"、逐步吸引这五类用户的过程。渗透率 15-20% 对应创新者和早期采用者（合计约 16%）被充分覆盖的临界点，行业进入快速"破圈"的成长期；渗透率 35-40% 是广发证券研究框架中的关键分界区间——在此区间内行业增速开始系统性放缓，从"高速增长"切换为"中速增长"。**注意**：此时早期大众仅覆盖约一半，该阈值反映的是**增速拐点**而非人群覆盖终点，不可与创新扩散理论机械对应。

### 增量 vs 存量渗透率

对于汽车等替换周期长的行业（约 10 年），应使用**"新车渗透率"**（当年销量中新能源占比）而非"保有量渗透率"（总保有量中新能源占比）。存量渗透率变化滞后期长，会严重低估行业发展阶段。

### 四种现实偏离

行业未必走完标准S形曲线，现实中可能发生四种偏离：

| 偏离类型 | 发生阶段 | 说明 | 示例 |
|---------|---------|------|------|
| **创业失败** | 导入期 | 商业模式根本跑不通，行业夭折 | 共享篮球、Clubhouse |
| **规模不及预期** | 成长期 | 天花板比预想的低，增长提前停滞 | 部分AI落地场景 |
| **稳定市场周期化** | 成熟期 | 不进入衰退，而是在成熟期内经历产能/库存周期波动 | 银行、白酒、空调 |
| **第二增长曲线** | 成熟期 | 开辟新市场/新技术，重新从导入期开始 | TikTok出海、扫地机器人 |

## A1 — 书中的应用 (Past Application)

### 案例 1: 智能手机渗透率验证两个临界点
- **问题**: 渗透率 15-20% 和 35-40% 这两个临界点是否可信？
- **方法论的使用**: 回测智能手机产业——2007年初代iPhone发布（导入期），2010年iPhone 4发布后渗透率突破20%（进入成长期），2011-2012年渗透率从22%飙升至40%，2014年渗透率超65%后行业明显进入成熟期
- **结论**: 两个临界点在智能手机产业中得到完整验证
- **结果**: 渗透率超过65%后行业整体估值中枢下移

### 案例 2: 新能源汽车渗透率——存量 vs 新车渗透率的陷阱
- **问题**: 2024年6月存量新能源汽车仅占汽车总保有量的7.18%，据此判断行业处于导入期合理吗？
- **方法论的使用**: 汽车替换周期约10年，应看新车渗透率。2022年新车渗透率稳定20%以上，2023年11月突破40%
- **结论**: 如果机械套用存量渗透率7.18%会误判为导入期，实质已是成长期
- **结果**: 正确判断阶段使分析重点从可行性切换为规模性+竞争格局

### 案例 3: 白色家电进入成熟期——格力美的双寡头格局
- **问题**: 白色家电行业进入成熟期后竞争格局如何演变？
- **方法论的使用**: 白色家电渗透率趋于饱和，进入成熟期后"卷"到极致，大量中小玩家被淘汰，格局从分散走向格力和美的的双寡头
- **结论**: 成熟期行业研究核心是护城河分析和竞争格局分析
- **结果**: 双寡头格局下新进入者难以撼动

## A2 — 触发场景 (Future Trigger) ★

### 语言信号

- "这个行业现在处于什么阶段？"
- "渗透率才X%，还有空间吗？"
- "这个行业增长是不是到头了？"
- "为什么同一个行业几年前估值100倍，现在只有20倍？"
- "现在入场会不会太晚了？"
- "这不是夕阳行业吗，还能投吗？"

### 与相邻 skill 的区分

- 与 `business-model-feasibility` 的区别: 定位术回答"行业在哪个阶段、该分析什么"，可行性验证回答"在导入期，这个具体模式能不能跑通"。前者是导航，后者是定点深挖。
- 与 `market-size-estimation` 的区别: 定位术帮你判断渗透率是否接近临界点，市场规模测算帮你具体量化TAM/SAM/SOM。

## E — 可执行步骤 (Execution)

1. **定位渗透率**
   - 得出行业当前渗透率的具体数值（来源：行业研报、协会数据、招股书）
   - 核对渗透率口径——是存量渗透率还是增量/新车渗透率？替换周期超过5年的行业必须使用增量渗透率
   - 搜索执行参考: `reference/search-specs.md`
   - 数据标注参考: `reference/data-annotation-standards.md`
   - 判停条件: 若无法获得任何可靠的渗透率数据，跳到步骤2用辅助信号辅助判断

2. **判断生命周期阶段**
   - 对标两个临界点（15%~20%和35%~40%），确定行业属于哪个阶段
   - 辅助信号交叉验证：行业是否在炒作新概念？产品是否已标准化？是否出现价格战？毛利率趋势？
   - 完成标准: 给出明确的阶段判断 + 渗透率计算依据 + 至少1个辅助信号

3. **评估偏离可能**
   - 逐一检查四种偏离可能性（创业失败/规模不及预期/稳定周期化/第二增长曲线）
   - 完成标准: 对每种偏离给出"可能/不太可能/需观察"的判断 + 依据

4. **输出该阶段的分析重点矩阵**
   - 基于阶段判断，明确各模块的分析权重（★★★/★★☆/★☆☆/☆☆☆）
   - 如果行业特性需要调整权重，明确说明调整原因
   - 完成标准: 输出优先级矩阵 + 对orchestrator的权重建议

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 用户问的是"某公司值不值得买"——公司分析需要叠加管理团队、财务指标等微观层面
- 用户已经明确了行业阶段且只是想确认一个具体数据——这是数据查询
- 行业本身就是极度成熟且无明显变化的基础行业（如自来水供应）——生命周期定位在这些行业中的边际价值很低

### 作者在书中警告的失败模式

- **用营收增速代替渗透率判断阶段**：口罩行业疫情期间增速激增不代表进入成长期，一次性供需错配不能替代渗透率的结构性判断
- **对所有行业机械套用同一种渗透率口径**：汽车行业10年替换周期，存量渗透率7%但新车渗透率已超40%，刻舟求剑会导致严重误判
- **面面俱到平均用力**：在导入期花大量时间分析竞争格局、在成熟期花大量时间论证需求真实性——都是资源错配

### 作者的盲点 / 时代局限 (来自 BOOK_OVERVIEW.md)

- **渗透率临界点的普适性未经验证**: 15%~20%和35%~40%的临界点来自智能手机等消费品经验，对To B行业、基础设施行业、周期性行业的适用性存疑
- **产业生命周期必然走完S形的假设**: 作者默认行业走完全程，但有些行业可能跳过某些阶段或长期停滞
- **A股中心视角**: 不同资本市场对生命周期的估值映射逻辑不同

### 容易混淆的邻近方法论

- **Gartner 技术成熟度曲线**: 描述的是单一技术（而非行业）的期望膨胀到幻灭再到复苏的周期。技术≠行业，不要用Gartner曲线判断行业阶段。
- **产能周期判断**: 产能周期是成熟期"稳定市场周期化"阶段内的**子周期**，不能替代产业生命周期定位。一个行业可能在成熟期内经历多轮产能周期。
