# 景气度追踪体系 — 验证中长期判断的高频工具

## R — 原文 (Reading)

> 景气度是宏观经济或某一行业的繁荣程度，用一篮子指标刻画的"体温计"。景气度追踪的目的是验证之前的分析假设，及时发现偏差，动态修正。
> 景气度是验证工具而非预测工具——它不能告诉你未来会怎样，但能告诉你"之前的判断有没有出问题"。
>
> — 肖璟, 第9章

---

## I — 方法论骨架 (Interpretation)

景气度追踪 = 用高频指标验证中长期判断。核心逻辑：**行业分析的结论需要通过可观察、可量化的指标来持续验证**。

### 追踪框架的两个维度

**纵向（产业链追踪）**：
- 上游：原材料价格、产量、开工率
- 中游：出厂价、库存量、产能利用率
- 下游：终端销量、零售价格、消费者信心

**横向（重点公司追踪）**：
- 季度营收/利润增速、毛利率变化
- 订单/合同负债（前瞻指标）
- 资本开支变化（CAPEX趋势）

### 验证/证伪条件清单

每个中长期判断必须配套"如果发生X，说明判断正确；如果发生Y，需要修正"的验证条件：

```
判断："行业处于成长期，渗透率将从25%升至40%"
  ✅ 验证信号：月度销量增速 > 30%、新用户增长 > 20%
  ❌ 证伪信号：连续3个月销量增速 < 15%、竞品渗透率开始下滑
```

### 景气度与产业链全景图联动

景气度指标的选择高度依赖于行业特性——不同行业的"体温计"完全不同：
- 制造业 → 开工率、PMI、出厂价格指数
- 消费 → 社零增速、客单价、同店增长
- 科技 → 研发投入占比、新产品发布节奏、客户采用率
- 周期 → CAPEX/折旧比、库存天数、分析师评级变化

## A1 — 书中的应用 (Past Application)

### 案例 1: 新能源汽车月度销量追踪
- **问题**: 如何持续验证新能源汽车行业的成长期判断？
- **方法论的使用**: 月度销量是核心景气度指标——如果连续多月销量增速保持在30%+，则成长期判断不变；如果增速连续放缓至15%以下，可能需要重新评估是否正在向成熟期过渡
- **结论**: 中汽协/乘联会月度数据是最可靠的追踪来源，可以在季度级别调整行业判断
- **结果**: 2023年价格战后，销量增速从50%+下降至30%左右，行业从成长期高速增长阶段进入成长期后期

### 案例 2: 经销商库存系数追踪汽车行业
- **问题**: 如何判断汽车行业是否面临供过于求？
- **方法论的使用**: 中国汽车流通协会每月发布"经销商库存系数"（期末库存量÷当期销售量）——系数≥2.5时，经销商经营压力大，车企产能短期不会扩张
- **结论**: 库存系数是判断汽车行业短期供需的最有效单一指标
- **结果**: 库存系数上升期间，整车企业股价通常承压

## A2 — 触发场景 (Future Trigger) ★

### 语言信号

- "怎么持续跟踪这个行业？"
- "有什么高频指标可以看？"
- "我怎么知道之前的判断还是对的？"
- "景气度怎么样？"
- "开工率在哪里查？"
- "怎么建立追踪体系？"

### 与相邻 skill 的区分

- 与 `external-force-scanner` 的区别: 外部扫描关注**中长期结构变量**（3-5年，如政策/人口/技术），景气度关注**高频交易指标**（月度/季度，如销量/价格/库存）。政策变化=外部扫描；月度销量=景气追踪。
- 与 `capacity-cycle-judgment` 的区别: 产能周期是**一次性的周期定位**，景气度是**持续性的验证追踪**。产能周期告诉你现在在哪，景气度告诉你在往哪走。

## E — 可执行步骤 (Execution)

1. **筛选核心景气度指标**
   - 纵向：沿产业链（上游/中游/下游）各选出1-3个最有代表性的指标
   - 横向：选出行业内TOP3-5企业的季度经营数据
   - 搜索执行参考: `reference/search-specs.md`
   - 完成标准: 至少3个高频指标 + 明确的数据来源和更新频率

2. **建立验证/证伪条件**
   - 为关键中长期判断（生命周期阶段、集中度趋势、估值判断）各设定1-2个验证条件
   - 完成标准: "验证/证伪"条件清单

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- 用户只需要一次性判断，不需要建立持续追踪体系——此时用其他子技能做单次分析即可
- 极度新兴、没有公开高频数据的行业——没有指标可以追踪
- 用户把景气度当作预测工具——景气度是验证工具，不是"景气度上升=股票会涨"

### 作者在书中警告的失败模式

- **指标太多导致噪音**：景气度追踪不是指标越多越好——每个行业只有3-5个真正有效的核心指标。过多的指标反而会产生矛盾的信号，让人无法判断
- **景气度滞后性**：很多统计指标有1-2个月的发布滞后期，等到景气度指标确认恶化时，股价可能已经跌了很多

### 作者的盲点 / 时代局限 (来自 BOOK_OVERVIEW.md)

- **A股数据生态中心**：书中引用的景气度指标（PMI、社零、行业协会数据）高度依赖中国数据发布体系。对于关注海外市场的投资者，需要找到对应市场的等价指标
- **另类数据的缺失**：书中未涉及卫星图像、招聘信息、信用卡消费等另类数据——这些在AI时代正成为越来越重要的景气度追踪手段
