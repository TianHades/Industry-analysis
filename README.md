# 行业快速分析 Skill

基于《如何快速了解一个行业》（肖璟著）方法论构建的 Claude Code Skill，通过系统化框架输出身份导向的行业分析报告。

## 特点

- **身份导向分析**：根据投资者/创业者/求职者/研究者等不同角色，输出针对性的分析报告
- **信源透明**：所有数据标注来源，区分实际数据/预测数据/不确定数据
- **AI能力边界透明**：明确标注 AI 可独立完成、需人工验证、需专业工具的内容
- **标准化框架**：涵盖行业定义、生命周期、市场规模、护城河、竞争格局等十大模块

## 快速开始

### 安装

将 [SKILL.md](SKILL.md) 复制到项目的 `.claude/skills/行业快速分析.md`（已在项目中完成此操作），Claude Code 会自动识别。

如果你在其他项目中使用，复制本项目的 `.claude/skills/行业快速分析.md` 到目标项目的相同路径即可。

### 使用

```
/行业快速分析
```

然后按提示输入你的身份和要分析的行业名称，可以根据需要适当修改提示词。

### 示例

```
请帮我分析一下 人工智能 行业
我是投资者身份
```

## 分析框架

```
行业研究 = [可行性 × 规模性 × 防守性 × 盈利性] × 估值 × 动态追踪
```

| 模块 | 说明 |
|------|------|
| 行业定义与边界 | 明确分析范围 |
| 产业生命周期 | 通过渗透率判断发展阶段 |
| 商业模式 | 可行性分析 |
| 市场规模 | TAM/SAM/SOM 测算 |
| 护城河 | 资源垄断 & 网络效应 |
| 竞争格局 | CR3/CR8 集中度分析 |
| 估值逻辑 | 不同阶段匹配不同估值方法 |
| PEST分析 | 政治经济社会技术外部因素 |
| 景气度跟踪 | 行业景气程度监控 |
| 产业链结构 | 上中下游关系 |
